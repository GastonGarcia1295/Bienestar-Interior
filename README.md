# 🌿 Bienestar Interior - Landing Page de Alta Conversión

Este proyecto es una Landing Page profesional desarrollada para la comercialización del E-book **"Tu Intestino Sano: El Protocolo Definitivo"**. Combina un diseño visual inmersivo con estrategias de conversión para maximizar la venta de productos digitales de salud.

## 🚀 Demo en Vivo
Puedes ver la página funcionando aquí: 
👉 [VER LANDING PAGE](https://gastongarcia1295.github.io/Bienestar-Interior/)

*(Nota: Una vez creado el repo, asegúrate de activar GitHub Pages en la pestaña Settings para que este link funcione).*

## 🎯 Objetivo del Proyecto
Crear una experiencia web de alto impacto que guíe al usuario desde el punto de dolor (inflamación) hasta la solución definitiva, utilizando:
1. **Impacto Visual:** Hero section con fondo temático desvanecido.
2. **Interactividad:** Animaciones de entrada y efectos de scroll.
3. **Urgencia y Acción:** Botones dinámicos con rebote y destellos rítmicos.
4. **Prueba Social:** Cinta transportadora continua de testimonios.

## 🛠️ Tecnologías Aplicadas
* **HTML5 / CSS3**: Estructura moderna.
* **Tailwind CSS**: Estilizado ágil y responsive (optimizado para celulares).
* **Animaciones Avanzadas**: 
    * Efecto *Bounce-In* (caída con rebote) en el encabezado.
    * Cinta de testimonios de movimiento infinito (Marquee).
    * Botones "vivos" con pulso y efecto de brillo (*Shimmer*).
* **Integración de Pagos**: Checkout directo con Mercado Pago.

## 📁 Instrucciones de Despliegue
1. Crear el repositorio con el nombre `Bienestar-Interior`.
2. Subir el archivo `index.html` (el código ultra-dinámico).
3. Subir la imagen `image_0.png` a la raíz del repositorio.
4. Activar **GitHub Pages** desde `Settings > Pages`.
